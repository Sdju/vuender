Пользователь не знает Swift UI
Пользователь полностью владеет Vue 3 и Web-разработку (для пояснений и сравнений)

Проект: аналог Finder для macOS

Структура проекта:
Sources/vuender/ - основной код проекта
  Core/ - основные модели и сервисы
  Services/ - сервисы
  Views/ - представления
    Content/ - представления контента
    FileBrowser/ - представления файлового браузера
    Header/ - представления заголовка
    Footer/ - представления нижнего колонтитула
